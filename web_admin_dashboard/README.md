# 🌐 POS Admin Dashboard Web Application

## 📋 نظرة عامة

تطبيق ويب حديث لإدارة نظام نقطة البيع (POS) مع واجهة عربية احترافية.

## ✨ المميزات

### 🔐 المصادقة
- تسجيل دخول آمن
- واجهة عربية بالكامل
- حماية من الجلسات المنتهية

### 📊 لوحة التحكم
- **نظرة عامة**: إحصائيات مباشرة ورسوم بيانية
- **التحليلات**: رسوم بيانية تفاعلية للمبيعات
- **المخزون**: إدارة المنتجات والمخزون
- **العملاء**: إدارة قاعدة بيانات العملاء
- **المستخدمون**: إدارة صلاحيات المستخدمين

### 🎨 التصميم
- واجهة عربية RTL بالكامل
- تصميم متجاوب للشاشات المختلفة
- ألوان احترافية ومتجانسة
- رسوم بيانية تفاعلية باستخدام fl_chart

## 🚀 التشغيل

### المتطلبات
- Flutter 3.0+
- متصفح ويب حديث

### خطوات التشغيل

1. **تثبيت الاعتماديات**
   ```bash
   flutter pub get
   ```

2. **تشغيل التطبيق**
   ```bash
   flutter run -d chrome
   ```

3. **فتح التطبيق**
   - المتصفح سيفتح تلقائياً على
   - `http://localhost:3000`

## 🌐 النشر

### Firebase Hosting (موصى به)
```bash
# تثبيت Firebase CLI
npm install -g firebase-tools

# نشر
firebase deploy --only hosting
```

### GitHub Pages
```bash
# بناء
flutter build web --base-href "/pos-dashboard/"

# نشر على GitHub Pages
```

### استضافة محلية
```bash
# بناء
flutter build web

# تشغيل محلي
cd build/web
python -m http.server 8000
```
